# react-native-hyperwallet

React Native wrapper for Hyperwallet

## Installation

```sh
npm install react-native-hyperwallet
```

## Usage

```js
import Hyperwallet from "react-native-hyperwallet";

// ...

const result = await Hyperwallet.multiply(3, 7);
```
